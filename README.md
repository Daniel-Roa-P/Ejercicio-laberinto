# Ejercicio laberinto

el actual repositorio posee un código en Python el cual al hacer uso de la programación funcional junto y hacer uso de árboles n-arios será posible determinar si un laberinto nxn que se encuentra en un txt posee solución al estar conformado por 0 (espacios vacíos), 1(paredes), X(inicio)y Y(destino).
